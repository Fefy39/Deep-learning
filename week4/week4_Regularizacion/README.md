# Semana 4 – Regularización

## Objetivo
Comparar una red neuronal base frente a una red con Dropout para analizar el efecto de la regularización sobre el overfitting y la generalización.

## Método de regularización aplicado
Se aplicó Dropout en la capa oculta de la red neuronal.

## Comparación realizada
Se entrenó el mismo modelo con y sin Dropout, usando el mismo dataset, misma arquitectura base y mismos parámetros de entrenamiento.

## Resultado principal
El modelo con regularización mostró un comportamiento más estable en validación y permitió analizar el efecto del Dropout sobre la generalización.
